================================================================================
                              HOW TO USE
================================================================================

1. OPEN: Open index.html in a modern web browser

2. INPUT: Choose one of three options:
   - Paste text directly into the textarea
   - Drag and drop a file (.txt, .docx, .pdf)
   - Click "Browse Files" to select a file

3. PROCESS: Click "Split into Rows" button

4. REVIEW: Check the output table for correct splitting

5. EXPORT: Choose one of:
   - Click "Copy Output" for clipboard
   - Click "Export" to download as Text, CSV, or JSON
